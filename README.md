# СБИС ↔ Аптеки — матчинг счетов-фактур

Решение кейса Simulative: сопоставление выгрузки из СБИС
с аптечными накладными и формирование Excel-файлов для 1С.

## Структура папок
sbis-apteka-matcher/
├── main.py
├── requirements.txt
├── README.md
├── Входящие/ ← сюда csv из СБИС
├── Аптеки/
│ └── csv/
│ └── correct/ ← сюда csv-файлы аптек
└── Результат/ ← создаётся автоматически


## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

Результаты появятся в `Результат/<ДД.ММ.ГГГГ>/`.

## Логика

1. Все csv из `Входящие` объединяются в один датафрейм СБИС.  
2. Столбцам присваиваются имена согласно заданию (snake_case).  
3. Для каждого csv-файла аптеки:
   - Если поставщик **ЕАПТЕКА** — к номеру накладной добавляется `/15`.
